# HTML: Page Structure

### Basic Page Structure
```
<html>
    <head>
        <title>
        </title>
    </head>
    <body>

    </body>
</html>
```

### HTML5 page structure

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
